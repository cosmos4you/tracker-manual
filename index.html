Яндекс Трекер — мануал для команды

Как пользоваться Яндекс Трекером

Три правила, которые заменяют весь мануал:
  1. Нет задачи в Трекере — нет задачи. Договорились в чате или голосом — сразу карточка.
  2. У задачи всегда есть дедлайн и тег клиента.
  3. Статусы честные: взял — «В работе», сделал — «Закрыт».

1Куда ставить задачу

ОчередьЧто сюдаОтветственный
SMMСоцсети, брифы, сметы, презентации, eventАлина
Продакшн (PROD)Съёмки, монтаж, дизайн, производство контентаАлина
ORMРепутация, отзывы, SERMНастя
PRСМИ, публикации, пиарНастя
Инфлюенс (INFL)Блогеры, интеграцииНастя

Не указала исполнителя — задача сама упадёт на Алину или Настю, они распределят. Ошиблась очередью — не страшно: меню «…» в задаче → «Переместить».

2Как поставить задачу

  1. ➕ (синяя кнопка слева) → «Задачу»
  2. Выбери очередь — по таблице выше
  3. Название глаголом: «Подготовить смету SMM для клиента Х»
  4. Описание: что сделать, что считается «готово», ссылки на материалы
  5. Дедлайн + тег клиента (например клиент-х) → «Создать»
Пример: «Подобрать 15–20 бьюти-блогеров 50–150К для клиента Х. Результат — таблица с охватами, ценой, контактом. Бриф: [ссылка]. Дедлайн 10.07, черновик 7.07». Всё, человеку не нужно переспрашивать.

3Клиенты и проекты: как структурируем

Выбирай инструмент по масштабу. Не поднимайся на уровень выше, пока не тесно на текущем:

МасштабИнструмент
Обычная задачаЗадача в очереди + тег клиента. 90% случаев.
Задача с этапами (например, организовать съёмку)Задача + подзадачи или чек-лист внутри неё
Комплексный запуск: несколько направлений сразу (SMM + продакшн + блогеры)Проект — к нему привязываются задачи из разных очередей, свой дедлайн и прогресс. Создаётся через ➕ → «Проект»
Крупный клиент с несколькими запускамиПортфель — «папка» проектов клиента: запуск, SERM-сопровождение, event и т.д. Раздел «Проекты и портфели» в левом меню

Тег клиента ставим всегда, на любом уровне: один фильтр по тегу — и видно всё по клиенту во всех очередях.

4Жизнь задачи

Открыт→ В работе→ Закрыт

  • Взял задачу — переведи «В работе» (кнопка вверху карточки). Сделал — «Закрыт».
  • Обсуждение — в комментариях задачи, не в личках. @имя — позвать человека.
  • Не успеваешь к дедлайну — напиши в комментарии заранее и предложи новый срок.
  • Файлы — перетащи в задачу; тяжёлое — на Диск, в задачу ссылку.

5Как ничего не пропустить

  • «Мои задачи» (значок ☑ слева) — твоя страница на каждый день: 5 минут утром на дедлайны и ответы.
  • Письма на почту приходят автоматически: новая задача на тебя, комментарий, упоминание. Не приходят — шестерёнка слева внизу → Настройки → Уведомления → включи канал «Почта».
  • Алине и Насте: подпишись на свою очередь (настройки очереди → «Уведомления» → «Задача создана»), чтобы видеть все новые задачи, а не только свои.
  • Доски — канбан для планёрок: карточки по колонкам статусов, двигаются мышкой. ➕ → «Доску».

6Значки левого меню

🔍Поиск — задача по номеру (SMM-5) или названию
➕Создать — задачу, очередь, доску, проект
☑Задачи — «Мои задачи», всё твоё
🗂Проекты и портфели — комплексные запуски (раздел 3)
🎯Цели — не используем
◇◇◇Очереди — все задачи направления
⊤⊥Доски задач — канбан
⊞Дашборды — статистика, пригодится позже
🕘История — недавно открытые задачи

Самый первый значок (▯) разворачивает панель в меню с подписями. На каждый день хватает трёх: ➕, ☑ и 🔍.
